# Othello
人vsコンピュータのオセロゲームのプログラム.
αβ法, minmax法でコンピュータは手を考える.

## 遊び方
`$ python3 run.py` でゲームはスタートする.

石を置きたい場所の行番号を `row >` に入力して `Enter` を押す. , 列番号を `col >` に入力して `Enter` を押す.
この繰り返し.

直近に置かれた石は点滅して表示される.
